Architecture

Three tiers. Zero waste.

Every task flows through a cascade of increasingly powerful solvers. Most never reach the API.

⚡

11 deterministic solvers: SymPy equations, arithmetic, date math, string ops, unit conversion, regex, factual lookup, number conversion, word analysis, sentiment.

Optimizations

13 token-saving innovations.

Every feature minimizes API tokens without sacrificing accuracy.

🔬 SymPy-LLM Symbiosis
LLM generates equation strings from word problems — SymPy solves locally with 100% accuracy. Zero hallucination risk.
📦 Session Dedup
Tasks sharing >80 character context batched into single API call. Saves 20-50% input tokens.
📐 Relevance Compression
TF-IDF sentence scoring keeps top 60% of content. Disabled for reasoning/math to preserve logical structure.
🪵 RTK Trace Compression
Truncates Python/JS stack traces to 10 lines. Saves 40-80% on debugging inputs.
✅ FrugalGPT Cascade
Validates Tier 1 via YES/NO judge. Auto-escalates to Tier 2 on failure. Guarantees accuracy gate pass.
🗳️ Self-Consistency Voting
3 parallel API calls for reasoning tasks. 2/3 majority consensus required.
🌡️ Temperature Scaling
Same-model fallback with different temperature (0.3) and injected mutation string for robustness.
⚡ Prompt Caching
Common prefix "HydraRoute | <category> |" — ~50% discount on cached tokens.
🎯 Per-Category max_tokens
Sentiment=4, factual=50, code=300, reasoning=300. No one-size-fits-all.
